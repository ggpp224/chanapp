# chanapp
> app 工程生成器及工具

## 安装
```
    $ npm install chanapp -g
```

## 使用

构建项目
```
    $ chanapp myApp
```

运行
```
    $ cd myApp
    
    $ npm install
    
    $ npm start
```

访问浏览器: ` http://localhost:8080/ `



tools:

生成React Component 模板文件
```
  $ chanapp create xx -c
```


生成 Page 模板文件
```
   $ chanapp create xx -p
```

`chanapp -h` 查看帮助
